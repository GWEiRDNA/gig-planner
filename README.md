# gig-planner
An app that helps you pick songs on your gig. Song organizer

DB Tables draft:
- Songs
    - Lyrics
    - Sheet Music
    - Midi files/MP3
- Song-Authors
- Authors
- Songs-Tags
- Tags
- Transition Graph
- Sets
- Tag Groups
- Events (For creating setlists for exact dates)
- Config
- Users
- Permissions
